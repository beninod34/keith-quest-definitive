# Keith Quest — Definitive Edition

A Pokémon fan game built on Pokémon Essentials, following the journey of **Keith** as he explores the region, battles trainers, and ultimately confronts a corrupted version of himself — **Shadow Keith** — in a climactic final battle.

This repository is the **Definitive Edition** of Keith Quest: a from-the-ground-up revision of the original project, migrating from Pokémon Essentials v19.1 to **v21** and reworking systems along the way.

## About

Keith Quest is a story-driven fangame with a full region to explore, a complete post-game, and a personal, character-focused narrative that builds toward a final confrontation with ... Tune in to find out!

## Features

- A complete main story plus an extended post-game with additional areas and minigames
- Trainer Card and full badge functionality
- Custom battle cinematics and character art during key story battles
- IV/EV viewing from the Summary screen, plus a post-game Nature/IV/EV modifier for build respec'ing
- Happiness-boosting items and quality-of-life NPC features (e.g. trading up rare candies)
- Custom moves, animations, and mechanics built around the story's climactic battles
- Built on a curated set of community plugins, including Elite Battle DX, Following Pokémon EX, HGSS Transitions, Fancy Badges, and more

## Definitive Edition Goals

This version aims to bring Keith Quest up to modern Pokémon Essentials standards:

- [ ] Upgrade the project from Pokémon Essentials v19.1 to **v21**
- [ ] Evaluate and integrate the Deluxe Battle Kit plugin
- [ ] Add a de-evolution item (obtained from Shadow Keith's husk after defeating him)
- [ ] General bugfixes, polish, and quality-of-life improvements carried over from the original release
